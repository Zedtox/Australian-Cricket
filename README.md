# Cricket Dynasty: Australia

## Overview

Cricket Dynasty: Australia is an interactive data visualisation dashboard that explores the history, dominance, and legacy of Australian cricket across international and domestic competitions.

The dashboard examines Australia's achievements through ICC tournaments, ICC rankings, Ashes rivalries, legendary players, influential captains, World Cup campaigns, and Big Bash League (BBL) performance. The project combines storytelling and interactive visualisation techniques to present over a century of Australian cricket history.

## Data Sources

### International Cricket Data

* ESPNcricinfo Statsguru
* Cricsheet

Used for:

* ICC rankings
* International match records
* Ashes history
* Player statistics
* Captaincy records
* World Cup history

### Domestic Cricket Data

* Kaggle Big Bash League Dataset

Used for:

* BBL franchise performance
* Championship wins
* Seasonal progression
* Geographic distribution of teams

## Interactive Features

* Format filtering for ICC rankings
* Zoomable ranking timeline using brushing
* Captain comparison selection
* BBL state filtering
* Team highlighting through legend interaction
* Animated BBL championship map
* Interactive tooltips across all visualisations
